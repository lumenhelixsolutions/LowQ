# LowQ Product Requirements

## Phase 1

- Single-file LowQ Lab.
- ROI Calculator MVP.
- Landing page.
- Certificate schema.
- License package.
- Pilot kit.
- Campaign assets.

## Phase 2

- LowQ Bridge local telemetry connector.
- Report PDF export.
- Certificate validator.
- API wrapper.
- Containerized proof runner.

## Phase 3

- LowQ Engine integration.
- scheduler adapters,
- model runtime adapters,
- cloud deployment package,
- enterprise dashboard,
- marketplace package.

## Non-functional requirements

- works out of the box,
- robust on long runs,
- modular adapters,
- no unsupported claims,
- deterministic proof exports,
- graceful failure states,
- local-first privacy by default.
