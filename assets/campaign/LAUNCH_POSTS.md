# LowQ Launch Posts

## GitHub launch
Introducing LowQ: an AI waste-heat reduction system that reduces compute waste before it becomes heat, then proves the delta.

Phase 1 includes a single-file proof lab, ROI calculator, certificate schema, pilot kit, and claim guardrails.

Run the workload. Measure the delta. Export the proof.

## Founder note
AI heat is not only a cooling problem. It is also a workload-design problem. LowQ starts upstream: fewer wasted tokens, less repeated context, less memory churn, better routing, and proof artifacts that separate estimates from measured evidence.

## Pilot CTA
We are opening a limited two-week LowQ Waste-Heat Assessment for teams running AI workloads with measurable power or telemetry traces.
